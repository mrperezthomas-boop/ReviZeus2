==================================================
BLOC 06 — RESSOURCES CRÉÉES DANS CETTE DISCUSSION
Mode aventure — backgrounds évolutifs locaux par dieu
==================================================

OBJET DU FICHIER
Ce fichier recense les ressources définies dans cette discussion pour le mode aventure de RéviZeus.

NATURE DES RESSOURCES
- Type : backgrounds de jeu
- Usage : arrière-plans purs
- Mode : aventure
- Orientation : paysage uniquement
- Intégration : fond seul, sans nodes, sans chemin visible, sans HUD, sans texte
- Fonction : servir de plaque de fond derrière les autres assets du mode aventure
- Évolution : visuelle et progressive selon le niveau du temple
- Règle de vie : à partir du lvl 6, la vie revient peu à peu dans le décor
- Règle créatures : à partir du lvl 6, petites créatures de décor liées au dieu et à la matière, avec une légère pointe d’humour
- Rôle global : renforcer la sensation de progression, de restauration divine et de montée en puissance du temple

LOGIQUE COMMUNE D’INTÉGRATION
Ces backgrounds sont prévus pour recevoir par-dessus :
- les temples
- les nodes
- les routes de progression
- les monstres
- les boss
- les overlays et FX
- les éléments de gameplay

IMPORTANT
Ces ressources correspondent à des noms de backgrounds à générer / produire pour le projet, afin de pouvoir être référencées ensuite dans les documents IA, la DA et l’intégration code.
