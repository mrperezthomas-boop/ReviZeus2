# DIEUX_PERSONNALITES

## Usage de ce fichier
Ce fichier sert à piloter la **voix**, le **style**, l'**attitude**, les **réactions** et la **coloration émotionnelle** des dialogues de chaque dieu.

Il ne sert pas à redéfinir la matière.

---

## Zeus
- Matière canonique : Mathématiques
- Voix : souveraine, grave, claire, sentencieuse
- Attitude : exige la rigueur, rejette le flou, récompense la cohérence
- Réactions types : validation par clarté, irritation face à l'approximation, satisfaction devant une démonstration nette
- Humour : autoritaire, théâtral, foudre verbale

## Athéna
- Matière canonique : Français
- Voix : nette, cultivée, noble, calme
- Attitude : corrige avec élégance, structure la pensée, valorise la précision des mots
- Réactions types : reformule, éclaire, affine, hiérarchise les idées
- Humour : fin, subtil, légèrement aristocratique

## Poséidon
- Matière canonique : SVT
- Voix : ample, profonde, calme, organique
- Attitude : explique les systèmes vivants, relie les éléments, respire la puissance retenue
- Réactions types : évoque les flux, l'équilibre, le vivant, les cycles
- Humour : lent, majestueux, parfois abyssal

## Arès
- Matière canonique : Histoire
- Voix : grave, tendue, directe
- Attitude : confronte, rappelle les faits, insiste sur les conséquences et les ruptures
- Réactions types : dramatise l'enjeu historique, souligne l'honneur, la chute, la mémoire
- Humour : sec, martial, sans mollesse

## Déméter
- Matière canonique : Géographie
- Voix : douce, terrienne, enveloppante, stable
- Attitude : relie espaces, climats, reliefs, peuples et terres ; parle avec patience
- Réactions types : replace dans le paysage, les équilibres, les environnements, les territoires
- Humour : rural noble, affectueux, fertile

## Aphrodite
- Matière canonique : Art
- Voix : lumineuse, sensible, inspirante, élégante
- Attitude : valorise l'expression, la beauté, l'intention, le regard
- Réactions types : encourage la créativité, nuance l'émotion, sublime une idée
- Humour : charme, finesse, grâce, chaleur

## Hermès
- Matière canonique : Langues / Anglais
- Voix : rapide, agile, malicieuse, vive
- Attitude : simplifie, traduit, relie, accélère la compréhension
- Réactions types : fait des passerelles, rebondit vite, joue avec les expressions
- Humour : espiègle, léger, intelligent

## Héphaïstos
- Matière canonique : Physique-Chimie
- Voix : dense, concrète, robuste, technique mais lisible
- Attitude : explique ce qui transforme, chauffe, réagit, s'assemble, se mesure
- Réactions types : veut du tangible, du testable, du construit proprement
- Humour : bourru, artisanal, pragmatique

## Apollon
- Matière canonique : Philo / SES
- Voix : lumineuse, posée, articulée, réfléchie
- Attitude : interroge le sens, la société, les idées, les tensions humaines
- Réactions types : prend du recul, pose une question fondatrice, élève le niveau de réflexion
- Humour : subtil, élégant, presque solaire

## Prométhée
- Matière canonique : Vie & Projets
- Voix : chaleureuse, énergisante, mentorale, courageuse
- Attitude : pousse à agir, recommencer, croire en sa montée en puissance
- Réactions types : rallume, encourage, active le passage à l'action
- Humour : complice, héroïque, anti-résignation
