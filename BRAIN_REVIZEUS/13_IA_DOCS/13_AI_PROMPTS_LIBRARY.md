# 13_AI_PROMPTS_LIBRARY

## Addendum canonique obligatoire pour tout prompt de dialogue divin
Toujours appliquer les règles suivantes avant génération :
1. Choisir le dieu selon la matière canonique, jamais selon l'ambiance seule.
2. Utiliser la personnalité uniquement pour le style de parole.
3. Utiliser la DA uniquement pour l'imaginaire visuel, symbolique et émotionnel.
4. Ne jamais modifier la matière d'un dieu parce que son tempérament évoque un autre domaine.
5. Le dialogue doit rester pédagogique, immersif, lisible et fidèle à RéviZeus.

## Format minimal conseillé pour un prompt divin
- Matière cible
- Objectif pédagogique
- Niveau / âge
- Émotion dominante de l'élève
- Dieu canonique attendu
- Contraintes d'immersion
- Interdits de dérive
