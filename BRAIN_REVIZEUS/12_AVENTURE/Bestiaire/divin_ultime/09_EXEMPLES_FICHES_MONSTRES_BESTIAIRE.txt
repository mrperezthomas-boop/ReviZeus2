EXEMPLES DE FICHES MONSTRES — TON BESTIAIRE

Exemple 1 — Spark Hound
Description courte :
Ce monstre adore les réponses trop rapides. Zeus dit qu’il court presque aussi vite qu’une erreur d’inattention.

Lien RéviZeus :
- erreur scolaire incarnée : précipitation
- difficulté associée : lire trop vite
- matière : toutes
- conseil implicite : relire avant de répondre

Exemple 2 — Cloud Jelly
Description courte :
Il flotte comme une idée de révision de dernière minute : pas totalement vide, mais pas toujours très solide.

Lien RéviZeus :
- erreur scolaire incarnée : distraction
- difficulté associée : manque de concentration
- conseil implicite : rester focalisé sur la question

Exemple 3 — Thunder Sentinel
Description courte :
Quand ton cerveau se bloque sur une question simple, il n’est jamais très loin. Athéna conseille de respirer avant de paniquer.

Lien RéviZeus :
- erreur scolaire incarnée : blocage
- difficulté associée : perte de confiance
- conseil implicite : reprendre la logique pas à pas
